Contexte métier connu à ce stade : le porc a une marge trop faible donc incompatible avec le coût de handling du hub Mâcon (R11) — seul le bœuf peut y transiter. L'économie éventuelle via le hub (bœuf uniquement) est estimée à environ 18 camions sur 2 mois (~2 camions/semaine), chiffre à confirmer avec l'analyse combinée Bigard + DPO.

Pour affiner les prévisions, il nous faut analyser les données Bigard et DPO ensemble — la simulation Hub Mâcon suivra après cette analyse des données groupées.
